## Dev Docs

### Install TypeORM
- npm install typeorm --save
- npm install reflect-metadata --save
- npm install @types/node --save-dev
- npm install pg --save
- npm install --save @nestjs/typeorm
- npm i dotenv
- npx typeorm init --name MyProject --database postgres


### Create Users Entiry
- nest g mo users
- nest g co users
- nest g s users
- nest g resource
